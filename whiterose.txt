Whiterose — v0.0.9
===================
C++17 CLI: git-aware safety shell.

Language: C++17, single file main.cpp.
Dependencies: libreadline (system library).
Install: make && sudo make install
Binary: ~/.local/bin/whiterose or /usr/local/bin/whiterose

Features
--------
- Boot guard — detects git repo root, tracks cwd, colorful prompt.
- Push auto-upstream — bare push auto-adds --set-upstream.
- Status reformat — color-coded staged/modified/untracked/conflicts.
- Force-push protection — blocks if remote has commits you don't.
- Protected branch guard — confirm before pushing to main/master/etc.
- Backup + undo — saves refs before destructive commands, restores on undo.
- Smart pull — rebase-if-safe on clean tree, merge fallback if dirty.
- Lockfile auto-fix — regenerates conflicted package-lock.json et al.
- Per-repo config — .whiterose.toml for customization.

Architecture
------------
C++ stays alive as main loop. Every command passes through C++ first.
fork() + execvp() into bash -i -c for execution. fd-3 pipe tracks cwd.
